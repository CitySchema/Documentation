Introducing the Repository Catalog

citySchema.org is a platform for sharing tools and templates for sharing a city-wide 3D model components among diverse tools including geographic information systems (GIS) and 3d modeling programs and tools for archival access and preservation. The citySchema Repository Catalog is a means of publishing a city model as a single package that includes all of the model components along with the catalog information, feature level metadata and finding aids in a single data-set that may be accessed with a variety of tools.

The design of the Repository Catalog is guided by the International Standards Organization Reference Model for Open Archival Information Systems (OAIS). OAIS is a convention for guiding information exchange activities so that the products are accessible and preservable over the long-term.

The Repository / Catalog is designed make it easy for users to access city model assets. The catalog repository can be easily be modified as a repository for city models covering other areas.

  • Take a tour of this full working demo created for the Boston Planning and Development agency.
  • Jump to the Documentation Index which explains a lot of the concepts about how we organize city models.
  • Or you can explore the same catalog as a simple file-system, which may be cloned from GitHub.

The documentation on the github repository has instructions for taking a self-guided tour that explains how the Reopsitory/Catalog can be modified using simple html and javascript skills to package publish your own city model.

In the coming days, this page will be expanded to provide more information on how your project can make use of the citySchema Repository Catalog to organize, share and preserve your city model assets.
