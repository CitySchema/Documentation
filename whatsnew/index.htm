What's New: Recent Updates and New Resources

  • January 12, 2022: Intro video posted on YouTube.

    The first video has been posted to the citySchema.org YouTube channel. Take and Give: Coordinating City Model Activity provides an intro to the citySchema architecture and a technical overview of the technique for sharing editable models with any commercial modeling tool while retaining precise 3D georeferencing.

  • January 1, 2022: A New Year of Funding for citySchema.org.

    The Boston Planning and Development Agency has renewed its support for citySchema.org with a new year of funding to support the continued development of tools and documentation!

  • December 27, 2021: Model Collection Management Tools for ArcGIS Pro posted on GitHub.

    An ArcGIS Pro project with all of the tools, tasks and layers necessary to demonstrate the procedures for updating the building model collection has been posted on GitHub. This repo has a demo data-set that canbe used to demonstrate all of the tasks, which are documented below in the section on Model Collection Management in ArcGIS Pro.
