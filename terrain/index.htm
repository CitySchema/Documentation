Modular Terrain Mesh

The citySchema tiling scheme includes detailed triangulated terrain meshes that snap together seamlessly. The terrain models extend to the corners of each tile. Inside the Boston city limit, the terrain model is based on Infotech 2011 DTM breaklines and masspoints. Areas beyond the city limit use the MassGIS 2005 DTM breaklines and masspoints.

The citySchema Modular-Terrain-Mesh repository contains resources and tools for creating new tiled terrain models in ArcGIS TIN format. The Clip Terrain Combo tool can be used to create a rectangular terrain model coverng any area within the Metro3D tile boundary. To assure that your model extends to the corners, you should snap the corners of your clip rectangle to points in the Metro3D_Tile_Pts feature class.

  1. Clone the workspace from git-hub CitySchema/Modular-TerrainRepository.
  2. Open the ArcGIS Pro project form the ArcDocs folder.

  4. Create a recangular clipping frame that is snapped to points in the Metro_3d_Tile_Pts feature class.
  5. From the Metro3d_Terrain toolbox, open the Clip Terrain_Combo model.
  6. Use the tools in the top row of the model to create a new folder to hold your terrain resources.
  7. Double-click the blue oval for Tile_Polys and substitute your new clipping frame.
